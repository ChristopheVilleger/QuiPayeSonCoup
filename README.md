Installation :  
- Composer install :  
`php composer.phar install`  
- Create Database :  
`php app/console doctrine:database:create`  
- Update schema :  
`php app/console doctrine:schema:update --force`
- Fixtures (will erase all data):  
`php app/console doctrine:fixtures:load`
